# alipay-sdk

自己在集成的时候踩了太多官方的坑了 = =

## 文件对于及说明

| 文件                                   | 说明     |
|:------------------------------------------|:-------|
| pc.php | 电脑网站支付 | 
| return.php                          | 同步回调通知 |
| notify.php                          | 异步回调通知 |
